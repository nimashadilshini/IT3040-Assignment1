# IT3040 Assignment 1 – Playwright Automation
Registration Number: IT23729148

## Project Description
This project contains automated test cases for testing a Singlish to Sinhala translation system (SwiftTranslator) using Playwright.

The tests cover:
- Positive functional scenarios
- Negative functional scenarios
- UI behavior
- Input robustness

## Technologies Used
- Node.js
- Playwright
- JavaScript

## Installation Steps

1. Install Node.js (LTS version)
2. Open terminal / command prompt
3. Navigate to the project folder
4. Run the following command:

   npm install

## Run Tests

To execute all tests:

   npx playwright test

To view HTML report:

   npx playwright show-report

## Browsers Tested
- Chromium
- Firefox
- WebKit
